# 🌱 KebunData DevOps

**KebunData** is a smart farming platform integrating IoT monitoring, Odoo ERP, and cloud automation.
This repository contains the Infrastructure as Code (IaC) and deployment pipelines to run the KebunData farm.

## 🚀 Features
* **IoT Integration:** Real-time monitoring (pH, Temp, VPD) via Raspberry Pi CM4 & Firebase.
* **ERP System:** Odoo 17 on Docker with PostgreSQL.
* **DevOps:** Automated CI/CD pipelines (GitHub Actions) & AWS backups.

## 📦 Quick Start
```bash
git clone [https://github.com/duap00/KebunData_DevOps.git](https://github.com/duap00/KebunData_DevOps.git)
